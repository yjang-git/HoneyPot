# What-If (시나리오 시뮬레이션) 테마

## 개요

**용도**: 제안 내용이 구현된 가상의 미래 장면을 시각화하여 설득력 있는 비전을 제시합니다.

**활용 상황**:
- "이 시스템이 도입되면 어떻게 바뀔까?"
- "우리 솔루션 적용 후 미래 모습"
- 연구 성과 적용 시나리오 시뮬레이션
- 정책/제도 변화 후 기대 효과

---

## 색상 팔레트

| 역할 | 색상 | HEX | 용도 |
|------|------|-----|------|
| 주조 | 미드나이트 틸 | #1A535C | 가상 장면의 프레임, 제목, 핵심 레이블 |
| 보조 | 슬레이트 블루 | #4ECDC4 | 시나리오 요소, 연결선, 프로세스 흐름 |
| 강조 | 선셋 오렌지 | #FF6B35 | 핵심 변화 포인트, 성과 지표, 하이라이트 |
| 배경 | 소프트 크림 | #F7FFF7 | 전체 배경, 미래적이면서 부드러운 느낌 |

**팔레트 특징**:
- 틸(청록)계열 주조로 미래지향적이면서 신뢰감 부여
- 오렌지 강조색이 "변화/개선" 포인트를 자연스럽게 강조
- 따뜻한 배경으로 위협적이지 않은 긍정적 미래 분위기

---

## 권장 레이아웃

### Section-Flow (최우선 권장)
- **상단**: 가상 시나리오의 메인 비주얼 (구현된 장면)
- **하단**: 구현까지의 단계 또는 핵심 효과 3-4개
- **장점**: "비전 + 실행 경로"를 한 장에 담을 수 있음

```
┌─────────────────────────────┐
│    [미래 구현 장면 비주얼]    │
└─────────────────────────────┘
     [단계 1] -> [단계 2] -> [단계 3]
```

### Z-Pattern (프레젠테이션용)
- **(1) 좌상**: 문제/현재 상태
- **(2) 우상**: 핵심 수치/기대 효과
- **(3) 좌하**: 구현 방법 요약
- **(4) 우하**: 미래 비전/CTA
- **장점**: 시선 흐름을 따라 자연스럽게 설득

```
(1) 현재 문제 ─────────────── (2) 기대 효과 +35%
        \
         \
(3) 솔루션 개요 ─────────────── (4) 구현된 미래
```

## 스타일 노트

- **장면 중심**: 추상적 도형보다 구체적인 상황/장면 묘사가 효과적
- **오버레이 텍스트**: 장면 위에 핵심 수치나 효과를 오버레이로 배치
- **시간축 암시**: 현재→미래 방향의 흐름을 시각적으로 표현
- **긍정적 톤**: 밝고 희망적인 분위기 유지 (어둡거나 위협적 요소 배제)
- **구체성**: "더 좋아진다" 대신 "30% 개선", "5분 단축" 등 수치화

---

## 적합한 케이스

- 신규 시스템/플랫폼 도입 제안
- 연구과제 기대 효과 시각화
- 정책 변화에 따른 미래 시나리오
- 솔루션 적용 전후 비교 (미래 강조)
- 투자/지원 요청 시 비전 제시

---

## 부적합한 케이스

- 현재 상태 분석에 집중하는 경우
- 복잡한 기술 아키텍처 설명
- 과거 데이터/트렌드 분석
- 비교 대상이 2개 이상인 경우 (→ comparison 테마 사용)
- 단계별 프로세스가 5개 이상인 경우 (→ gov/seminar 테마 사용)
