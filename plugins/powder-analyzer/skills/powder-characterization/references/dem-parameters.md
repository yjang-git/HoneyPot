# DEM 파라미터 문헌값 데이터베이스

## 파라미터 3-분류 체계: Measured → Derived → DEM

```
🔴 직접 측정 (Measured)          🟣 유도 계산 (Derived)          🔵 DEM 입력 (Calibrated)
─────────────────────          ──────────────────────          ──────────────────────
ρ_b (겉보기밀도)     ──────→   HR = ρ_tap / ρ_b      ──────→   (유동성 판정 기준)
ρ_tap (탭밀도)       ──────→   CI = (1-ρ_b/ρ_tap)×100
t_50 (Hall/Carney)             ff = σ₁/f_c           ──────→   (호퍼 설계 기준)
AoR (안식각)         ──────────────────────────────────────→   μ_r (구름마찰) ← AoR 역해석
m(t) (호퍼 유량)     ──────→   W, C, k (Beverloo)    ──────→   (오리피스 설계)
(σ,τ) 전단 데이터    ──────→   φ_i (내부마찰각)       ──────→   μ_pp = tan(φ_i)
                              c (점착력)              ──────→   γ_JKR (표면에너지)
(σ_w,τ_w) 벽면 데이터 ─────→   φ_w (벽면마찰각)       ──────→   μ_pw = tan(φ_w)
                              f_c (일축압축강도)
```

### 교정 대상 매핑표

| 실험 측정값 | 유도 파라미터 | DEM 입력 | 교정 방법 |
|------------|-------------|---------|-----------|
| (σ,τ) 전단 | φ_i, c | μ_pp = tan(φ_i) | Jenike D6128 |
| (σ_w,τ_w) | φ_w | μ_pw = tan(φ_w) | D6128 §7.6 |
| AoR 원추 | AoR (°) | μ_r | 안식각 역해석 |
| 전단 c | f_c | γ_JKR | JKR 역해석 |
| 문헌값 | E, ν | E*, ν | 나노인덴테이션/문헌 |
| 낙하시험 | e | e | 직접 또는 문헌 |
