# ASTM 표준 8종 요약 비교표

## 밀도 측정 표준

### B329 - Scott Volumeter 겉보기 밀도
- **장비**: Scott Volumeter (4개 유리 판, 25 cm³ 컵)
- **시료량**: 25 cm³ 컵이 가득 찰 때까지
- **적용**: 전 입도 범위, 특히 미세 분말에 적합
- **계산**: ρ_apparent = m / 25 (g/cm³)
- **반복성**: ±0.04 g/cm³

### B417 - Carney Funnel 겉보기 밀도
- **장비**: Carney Funnel (오리피스 5.08 mm)
- **시료량**: 25 cm³ 컵이 가득 찰 때까지
- **적용**: B329보다 넓은 오리피스로 유동 불량 분말에 대안
- **계산**: ρ_apparent = m / 25 (g/cm³)

## 유동성 측정 표준

### B213 - Hall Funnel 유동 속도
- **장비**: Hall Funnel (오리피스 2.54 mm, 반각 30°→원추각 60°)
- **시료량**: 50 g
- **절차**: 손가락으로 막은 후 시료 투입 → 개방 → 전량 통과 시간 측정
- **결과**: 초/50g 또는 "Did not flow"
- **주의**: D50 < 10 μm 미세 분말은 높은 확률로 No Flow
- **오리피스 공차**: Ø2.54 mm ± 0.04 mm (핀게이지로 검증)
- **재질**: 스테인리스강, 내벽 연마 (Ra < 0.8 μm)
- **Empty Annulus**: 유효 직경 = D_o − k·d_p. NMC811(D50 ~5μm)에서 k·d_p ≈ 7μm → 무시 가능
- **환경**: 23 ± 2°C, 50 ± 5% RH (§7.1)

### B964 - Carney Funnel 유동 속도
- **장비**: Carney Funnel (오리피스 5.08 mm, 원추각 60°)
- **시료량**: 150 g (B213보다 많음)
- **적용**: B213에서 유동 불가한 분말의 대체 측정
- **결과**: 초/150g
- **오리피스 공차**: Ø5.08 mm ± 0.05 mm
- **재질**: 스테인리스강, 내벽 연마 (Ra < 0.8 μm)

## 전단 측정 표준

### D6128 - Jenike Shear Test

- **장비**: Jenike Shear Tester (내경 95.25 mm 원형 셀)

#### Fig.4 정밀 치수

| 부품 | 치수 | 허용 범위 |
|------|------|-----------|
| Base 높이 | 12.7 mm (0.5 in) | ±0.1 mm |
| Ring 높이 | 15.875 mm (0.625 in) | ±0.1 mm |
| Mold ring 높이 | 9.525 mm (0.375 in) | ±0.1 mm |
| 셀 내경 | 95.25 mm (3.75 in) | ±0.1 mm |
| 벽 두께 | ≥ 3 mm | 강성 확보 |
| Lid 직경 | 93.3 mm | ±0.2 mm |
| 셀 단면적 A | 7,126 mm² | ±1% |

#### §8 전단 시험 절차

1. **시료 준비**: Mold ring + Ring + Base를 조립, 분말 충전, Mold ring 제거
2. **예비 전단 (Pre-shear)**: σ_pre 하중 하에서 정상 상태(τ 일정)까지 전단
3. **전단 (Shear)**: σ < σ_pre인 수직 응력에서 전단 → 최대 τ 기록
4. **반복**: 3~5개 σ 수준에서 반복 (하나의 YL에 3+ 포인트)
5. **항복 궤적 (YL) 작도**: (σ, τ) 점들에 직선 피팅
6. **Mohr 원 분석**: YL 접선인 대·소 Mohr 원에서 σ₁, f_c 도출
7. **전단 속도**: 2.72 mm/min (0.18 in/min), 허용 2.54–3.00 mm/min

#### §7.6 벽면 마찰 시험 절차

1. Base 대신 **쿠폰(벽면 재질 시편)**을 셀 하부에 설치
2. Ring에 분말 충전 (Mold ring으로 과충전 후 제거)
3. 최대 σ_w6에서 시작 → σ_w를 **감소 방향**으로 6단계 적용
4. 각 σ_w에서 τ_w 기록
5. τ_w vs σ_w → 기울기 tan(φ_w) = 벽면 마찰각

**전단 vs 벽면 마찰 차이:**

| 항목 | 전단 (§8) | 벽면 마찰 (§7.6) |
|------|-----------|-----------------|
| 바닥면 | Ring + Base | 쿠폰 |
| 전단면 | 분말↔분말 | 분말↔쿠폰 |
| 출력 | φ_i, c, f_c, ff | φ_w |
| 하중 | σ 고정, 각 포인트별 | σ_w 감소 (6→1) |

#### Annex A2: 하중 계산

```
W = m_lid × g + m_W × g     [N]
σ = W / A                    [Pa]

A = π/4 × (95.25)² = 7,126 mm² = 7.126 × 10⁻³ m²
```

- m_lid: 뚜껑 질량 (칭량 필수)
- m_W: 행거 + 추 질량
- σ_pre, σ는 추 교체로 조절

#### 힘 전달 경로

```
①로드셀 ← ②스템 ← ③브래킷 ← ④핀 ← Ring
```

- 로드셀: 전단력 F → 전기 신호
- τ = F / A

- **출력**: ff (Flow Function), φ_i (내부 마찰각), φ_w (벽면 마찰각), c (점착력)
- **설계 적용**: Jenike 설계 차트로 호퍼 반각 θ 결정

## 동적/투과 측정 표준

### D8327 - FT4 Permeability
- **장비**: FT4 Powder Rheometer
- **측정**: 공기 투과도 (압력 강하 vs 공기 유속)
- **용도**: 에어레이션 효과, 탈기 특성 평가

### D8328 - FT4 Dynamic Flow
- **장비**: FT4 Powder Rheometer (블레이드 회전)
- **측정값**:
  - BFE (Basic Flowability Energy): 기본 유동 에너지 (mJ)
  - SE (Specific Energy): 비에너지 (mJ/g)
  - SI (Stability Index): BFE 변동 지수
  - FRI (Flow Rate Index): 유속 의존성
  - CI (Compressibility Index): 1–15 kPa 압축성
- **용도**: 유동성 종합 지표, 공정 상관성 높음
