# 간이(Custom) 장치 사양서

ASTM 규격 장비 대신 자체 제작/간이 장치를 사용하는 경우의 사양, ASTM 원본 대비 차이, 허용 기준을 정의한다.

## Equipment Mode

| 모드 | 설명 | 적용 |
|------|------|------|
| **standard** | ASTM 규격 상용 장비 사용 | 기본값 |
| **custom** | 자체 제작 간이 장치 사용 | 사양 기재 필수 |

custom 모드 사용 시 아래 5가지 장치 각각에 대해 ASTM 대비 차이를 명시해야 한다.

## 장치 2: Hall/Carney 유동도계

### ASTM B213 (Hall Funnel) 규격

| 항목 | ASTM 규격 | 허용 범위 |
|------|-----------|-----------|
| 오리피스 직경 | 2.54 mm | ±0.04 mm |
| 원추 반각 | 30° (전체 60°) | ±0.5° |
| 시료량 | 50 g | ±0.1 g |
| 재질 | 스테인리스강, 내벽 연마 | Ra < 0.8 μm |

### ASTM B964 (Carney Funnel) 규격

| 항목 | ASTM 규격 | 허용 범위 |
|------|-----------|-----------|
| 오리피스 직경 | 5.08 mm | ±0.05 mm |
| 원추 반각 | 30° (전체 60°) | ±0.5° |
| 시료량 | 150 g | ±0.1 g |
| 재질 | 스테인리스강, 내벽 연마 | Ra < 0.8 μm |

### Empty Annulus 개념

오리피스 벽면 근처에서 입자가 없는 환형 공간:
- 유효 직경 = D_o - k·d_p (Beverloo의 k 항)
- Hall Funnel에서 D50 > Ø2.54/10 = 254 μm 이상이면 empty annulus 효과 무시 가능
- NMC811 (D50 ~5 μm) → k·d_p ≈ 1.4 × 5 = 7 μm → 오리피스 대비 무시 가능

### Custom 장치 체크리스트

- [ ] 오리피스 직경 핀게이지로 실측 (B213: 2.54±0.04, B964: 5.08±0.05)
- [ ] 원추 내벽 연마 상태 (해칭 없이 매끄러운 표면)
- [ ] 수집 컵: 25 cm³ ± 0.05 cm³ (Ø34 × H28 mm 참조)
- [ ] 깔때기 수직 설치 확인 (스탠드 + 수준기)

## 장치 3: Jenike 전단 셀 (D6128)

### ASTM D6128 Fig.4 정밀 치수

| 부품 | 치수 | 허용 범위 | 비고 |
|------|------|-----------|------|
| Base (하부 링) 높이 | 12.7 mm | ±0.1 mm | 0.5 in |
| Ring (상부 링) 높이 | 15.875 mm | ±0.1 mm | 0.625 in |
| Mold ring 높이 | 9.525 mm | ±0.1 mm | 0.375 in |
| 셀 내경 (ID) | 95.25 mm | ±0.1 mm | 3.75 in |
| 벽 두께 | ≥ 3 mm | - | 강성 확보 |
| Lid 직경 | 93.3 mm | ±0.2 mm | 셀 ID보다 ~2mm 작음 |
| 셀 단면적 (A) | 7,126 mm² | ±1% | π/4 × 95.25² |

### 전단 시험 조건

| 항목 | 값 | 출처 |
|------|-----|------|
| 전단 속도 | 2.72 mm/min (0.18 in/min) | D6128 §8.3 |
| 허용 범위 | 2.54–3.00 mm/min | ±10% |
| 전단 방향 | 수평 (→) | Ring을 Base 위에서 수평 이동 |
| 수직 하중 방향 | 연직 (↓) | Lid 위에 행거 + 추 |

### 힘 전달 경로 (D6128 §8)

```
①로드셀 ← ②스템 ← ③브래킷 ← ④핀 ← Ring
     ↑
  전단력 τ 측정
```

- 로드셀: 전단력 F를 전기 신호로 변환
- 스템: 수평 강체 봉 (Ring의 핀에 연결)
- 브래킷: 스템과 핀을 연결하는 L자 금구
- 핀: Ring 외벽에 삽입되어 수평력을 전달

### 하중 계산 (Annex A2)

```
σ = W / A

여기서:
  W = m_lid × g + m_W × g    [N]
  A = 7,126 mm² = 7.126 × 10⁻³ m²
  σ = W / A                    [Pa]
```

- m_lid: 뚜껑 질량 (고정값, 칭량 필수)
- m_W: 추 질량 (행거 질량 포함)
- g = 9.81 m/s²
- 예비 전단(Pre-shear) 응력 σ_pre와 전단 응력 σ는 추 교체로 조절

### 전단면 위치

- Ring과 Base의 경계면 = 전단면 (shear plane)
- 분말↔분말 전단 (내부 마찰)
- 전단 시 Ring이 Base 위에서 수평 이동 → 전단면에서 분말 파괴

### Custom 장치 체크리스트

- [ ] Base/Ring/Mold 높이 마이크로미터로 실측
- [ ] 셀 내경 95.25 ± 0.1 mm 확인
- [ ] 벽 두께 ≥ 3 mm 확인
- [ ] Lid 직경 93.3 ± 0.2 mm 확인 (셀 내벽에 자유 이동)
- [ ] 전단 속도 2.72 mm/min ± 10% 확인
- [ ] 로드셀 교정 확인 (0–50 N 범위)
- [ ] 행거 + 추 질량 0.01 g 단위로 칭량

## 장치 4: 벽면 마찰 시험 (D6128 §7.6)

### 장치 3과의 차이

| 항목 | 전단 시험 (§8) | 벽면 마찰 (§7.6) |
|------|---------------|-----------------|
| 바닥면 | Ring + Base (분말 충전) | 쿠폰 (벽면 재질) |
| 전단면 | 분말 ↔ 분말 | 분말 ↔ 쿠폰 |
| 측정값 | φ_i, c, f_c, ff | φ_w (벽면 마찰각) |
| 하중 방향 | σ 고정, τ 측정 | σ_w 감소 방향 (6→1) |

### 벽면 마찰 절차 (§7.6)

1. Base 대신 쿠폰(벽면 재질 시편)을 Jenike 셀 하부에 설치
2. Ring에 분말 충전
3. 최대 수직 하중(σ_w6)에서 시작
4. 전단하면서 σ_w를 단계적으로 감소 (σ_w6 → σ_w5 → ... → σ_w1)
5. 각 σ_w에서의 τ_w를 기록
6. τ_w vs σ_w 직선 피팅 → 기울기 = tan(φ_w)

### 쿠폰 재질 예시

| 재질 | μ_pw 예상 범위 | 용도 |
|------|---------------|------|
| STS304 | 0.2–0.4 | 호퍼/배관 표준 |
| Al6061 | 0.15–0.35 | 경량 호퍼 |
| PTFE | 0.05–0.15 | 최저 마찰 코팅 |
| 경질 크롬 | 0.1–0.25 | 내마모 코팅 |

### Custom 장치 체크리스트

- [ ] 쿠폰 표면 조도 Ra 측정 (조도계 또는 비교 시편)
- [ ] 쿠폰 평탄도 확인 (게이지로 ±0.05 mm)
- [ ] 스템 이동 한계 ≤ 6 mm 확인 (D6128 §8.3.4)
- [ ] σ_w 감소 방향 순서 준수 (최대 → 최소)

## 장치 5: 안식각/호퍼 배출

### 안식각 측정

| 항목 | 값 | 비고 |
|------|-----|------|
| 방법 | 고정 기저 원추법 | 평판 위 자유 퇴적 |
| 측정 | θ = arctan(2H/D) | H: 원추 높이, D: 기저 직경 |
| 반복 | 3회 | |
| 주의 | 일정 속도로 시료 인상 | 충격/진동 금지 |

### 호퍼 배출 시험

| 항목 | 사양 |
|------|------|
| 호퍼 재질 | STS304 (Ra < 0.8 μm) |
| 호퍼 반각 | 30° |
| 오리피스 세트 | 7종: 5, 8, 10, 12, 15, 20, 25 mm |
| 교체 방식 | 나사식 플레이트 또는 끼움식 |
| 게이트 | 수평 슬라이드 |
| 수집 | 전자저울 위 컵 (1초 간격 m(t) 기록) |

### 아칭 (Arching) 판단

- D < D_c 일 때 오리피스에 안정 아치(arch/bridge) 형성 → 유동 차단
- D_c = 임계 오리피스 직경 (실험으로 결정)
- Beverloo 방정식: W = C·ρ_b·√g·(D_o − k·d_p)^{5/2}
  - **C** (~0.58, 대문자): 방출 계수 — 오리피스 형상·재질에 의존
  - **c** (소문자): 점착력(cohesion) — 전혀 다른 물리량, 혼동 주의

### Custom 장치 체크리스트

- [ ] 오리피스 직경 핀게이지로 실측 (각 ±0.1 mm)
- [ ] 호퍼 내면 조도 Ra 확인
- [ ] 저울 분해능 0.01 g, 기록 주기 1초
- [ ] 게이트 밸브 개방 시 충격 최소화
