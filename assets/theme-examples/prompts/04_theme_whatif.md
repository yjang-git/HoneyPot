# INSTRUCTION

### Image Purpose
AI 스마트 팩토리가 완전히 구현된 미래의 일상 장면을 보여주는 비전 몰입형 슬라이드. 관객이 이미 성공한 미래 안에 서 있는 느낌을 준다.

### Target Audience
투자자, 경영진, 정책 결정자

### Key Message
AI 스마트 팩토리가 일상이 된 세상 — 이미 작동 중이다

### Visual Style
whatif 스타일 — 공상과학 영화 UI처럼 빛나는 HUD 속 미래가 작동 중이다. 단일 장면(After만). Before가 없다. 이미 After 안에 서 있다.

### Rendering Style
- **서피스**: 프로스티드 글래스 패널(rgba(255,255,255,0.08) 배경 + 1px rgba(255,255,255,0.15) 테두리 + backdrop blur). 내부 정보는 깨끗한 산세리프.
- **배경**: 다크 틸(#1A535C)에서 소프트 크림(#F7FFF7)으로의 대각선 그라데이션. 서브틀한 원형 보케 3-4개.
- **코너/엣지**: 큰 라운딩(16px radius). 일부 필(pill) 형태 허용.
- **연결선**: 그라데이션 라인(보조→강조) + 빛나는 글로우 효과. 흐르는 도트 라인 느낌.
- **시각 장식**: 반투명 원형 보케, 얇은 원형 링 장식, 파티클 도트, HUD/대시보드 UI 요소.
- **타이포그래피**: 제목은 대형 선언문 스타일. KPI 수치는 강조색(#FF6B35)으로 크게. 본문은 보조색(#4ECDC4).
- **공간 구성**: 단일 장면. 현재 시제 선언("이미 작동 중이다"). 프로스티드 글래스 패널 안에 정보 배치.

### Content Placement
- 상단 좌측에 '무인 스마트 팩토리가 가동 중입니다' 텍스트를 대형 선언문 스타일로 배치, 미래가 이미 현실임을 선포
- 화면 중앙에서 우측에 걸쳐 큰 프로스티드 글래스 패널 하나를 배치하고 내부에:
  - '실시간 가동률' 텍스트와 바로 아래 '98.7%' 텍스트를 강조색(#FF6B35)으로 크게
  - '일일 생산량' 텍스트와 바로 아래 '12,400대' 텍스트
  - '불량률' 텍스트와 바로 아래 '0.02%' 텍스트
- 좌측 하단에 작은 프로스티드 글래스 패널을 배치하고 '24시간 무중단 · AI 자율운영' 텍스트
- 배경의 대각선 그라데이션 위로 HUD 스타일의 얇은 원형 링과 파티클 도트가 장식적으로 흩어져 있다
- 우하단에 '지금, 이 순간에도' 텍스트를 소형으로 배치

# CONFIGURATION

### Canvas Settings
- 해상도: 3840 x 2160 (4K)
- 비율: 16:9
- 배경색: #1A535C (미드나이트 틸) 기반 대각선 그라데이션

### Background Treatment
좌상단 #1A535C(미드나이트 틸)에서 우하단 #F7FFF7(소프트 크림)으로의 대각선 그라데이션. 서브틀한 원형 보케 3-4개(rgba(78,205,196,0.15))와 얇은 원형 링 장식 2개. 파티클 도트 5-6개 흩뿌림.

### Color Palette
- 주조: #1A535C (미드나이트 틸) — 장면 프레임, 제목, 핵심 라벨
- 보조: #4ECDC4 (슬레이트 블루) — 미래 요소, 연결선, UI 하이라이트, 본문
- 강조: #FF6B35 (선셋 오렌지) — 핵심 성과 KPI 수치, 콜아웃
- 배경: #F7FFF7 (소프트 크림) — 밝고 희망적인 미래 분위기

### Layout Structure
Future Snapshot (미래 스냅샷) — 단일 장면 안에 프로스티드 글래스 패널로 정보를 자연스럽게 배치

### Typography
- 제목: 대형 선언문 스타일, #1A535C, 굵게
- KPI 수치: 초대형, #FF6B35, 굵게
- KPI 라벨: 중형, #4ECDC4
- 본문/캡션: 소형, #4ECDC4 또는 #1A535C
