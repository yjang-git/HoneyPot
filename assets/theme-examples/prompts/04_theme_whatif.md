# INSTRUCTION

### Image Purpose
AI 스마트 팩토리가 완전히 구현된 미래의 일상 장면. 관제실에서 관리자가 자율 운영되는 공장을 여유롭게 모니터링하는 보통의 하루를 보여준다. 관객이 이미 성공한 미래 안에 서 있는 느낌을 준다.

### Target Audience
투자자, 경영진, 정책 결정자

### Key Message
AI 스마트 팩토리가 일상이 된 세상 — 이미 작동 중이다

### Visual Style
whatif 스타일 — 공상과학 영화 UI처럼 빛나는 HUD 속 미래가 작동 중이다. 단일 장면(After만). Before가 없다. 이미 After 안에 서 있다.

### Rendering Style
- **서피스**: 프로스티드 글래스 패널(rgba(255,255,255,0.08) 배경 + 1px rgba(255,255,255,0.15) 테두리 + backdrop blur). 내부 정보는 깨끗한 산세리프.
- **배경**: 미래적인 관제실 환경. 대형 유리창 너머로 자동화된 공장 라인이 보인다. 다크 틸(#1A535C) 톤의 조명.
- **코너/엣지**: 큰 라운딩(16px radius). 일부 필(pill) 형태 허용.
- **연결선**: 그라데이션 라인(보조→강조) + 빛나는 글로우 효과. 흐르는 도트 라인 느낌.
- **시각 장식**: 반투명 원형 보케, 얇은 원형 링 장식, 파티클 도트, HUD/대시보드 UI 요소. 관제실 모니터 화면에 자연스럽게 통합.
- **타이포그래피**: 제목은 대형 선언문 스타일. KPI 수치는 강조색(#FF6B35)으로 모니터 화면 안에 자연스럽게 표시. 본문은 보조색(#4ECDC4).
- **공간 구성**: 단일 시네마틱 장면. 현재 시제. 관제실 내부에서 유리창 너머 공장을 바라보는 구도. 전경-중경-후경 3레이어 깊이감.

### Content Placement
- 화면 상단 좌측에 '무인 스마트 팩토리가 가동 중입니다' 텍스트를 대형 선언문 스타일로 배치
- 전경 좌측에 관리자 한 명이 편안한 자세로 의자에 앉아 커피를 들고 대형 모니터를 바라보고 있다. 여유로운 표정과 자세로 "이것이 일상"임을 암시한다
- 중경에 관리자 앞의 대형 곡면 모니터 2-3개가 있고, 모니터 화면 안에 HUD 스타일로 다음 정보가 자연스럽게 표시된다:
  - 첫 번째 모니터에 '실시간 가동률' 텍스트와 '98.7%' 텍스트가 오렌지색(#FF6B35)으로 크게 표시
  - 두 번째 모니터에 '일일 생산량 12,400대' 텍스트와 '불량률 0.02%' 텍스트가 표시
- 후경에 대형 유리창 너머로 자동화된 공장 라인이 보인다: 로봇 팔들이 정밀하게 작업 중이고, 컨베이어 벨트 위로 제품이 흘러간다. 틸 계열 조명이 공장 전체를 비추고 있다
- 관제실 천장과 모니터 주변에 프로스티드 글래스 패널과 HUD 요소(얇은 원형 링, 파티클 도트)가 미래적 분위기를 연출
- 좌측 하단에 작은 프로스티드 글래스 패널로 '24시간 무중단 · AI 자율운영' 텍스트
- 우하단에 '지금, 이 순간에도' 텍스트를 소형으로 배치

# CONFIGURATION

### Canvas Settings
- 해상도: 3840 x 2160 (4K)
- 비율: 16:9
- 배경색: #1A535C (미드나이트 틸) 톤의 관제실 환경

### Background Treatment
미래적 관제실 내부. 다크 틸(#1A535C) 톤의 간접 조명. 대형 유리창 너머로 자동화 공장 라인이 보케와 함께 보인다. 서브틀한 원형 보케 3-4개(rgba(78,205,196,0.15))와 얇은 원형 링 장식이 HUD 요소로 떠 있다.

### Color Palette
- 주조: #1A535C (미드나이트 틸) — 관제실 조명, 제목, 핵심 라벨
- 보조: #4ECDC4 (슬레이트 블루) — 모니터 UI, 연결선, HUD 하이라이트, 본문
- 강조: #FF6B35 (선셋 오렌지) — 핵심 KPI 수치(98.7%), 콜아웃
- 배경: #F7FFF7 (소프트 크림) — 유리창 너머 공장의 밝은 조명

### Layout Structure
Future Snapshot (미래 스냅샷) — 관제실에서 유리창 너머 공장을 바라보는 단일 시네마틱 장면. KPI는 모니터 화면 안에 자연스럽게 통합.

### Typography
- 제목: 대형 선언문 스타일, #F7FFF7(밝은 색), 굵게
- KPI 수치: 초대형, #FF6B35, 모니터 화면 안에 표시
- KPI 라벨: 중형, #4ECDC4, 모니터 화면 안에 표시
- 본문/캡션: 소형, #4ECDC4 또는 #F7FFF7
