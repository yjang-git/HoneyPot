# INSTRUCTION

### Image Purpose
스마트 팩토리에서 AI 에이전트들이 제조 공정을 자율적으로 운영하는 모습을 Kurzgesagt 풍 시각 스토리텔링으로 표현한 교육용 일러스트레이션

### Target Audience
기술 혁신에 관심 있는 일반 대중, 연구자, 정책 결정자

### Key Message
AI 에이전트들이 인간과 협력하여 공장을 자율적으로 운영하는 미래 제조 혁신

### Visual Style
concept 스타일 — Kurzgesagt 풍 플랫 벡터 일러스트레이션. 텍스트 없이 장면과 시각적 메타포만으로 개념을 전달하는 교육용 일러스트레이션.

### Rendering Style
- **서피스**: 플랫 벡터 일러스트레이션. 둥근 기하학적 형태와 부드러운 곡선. 각 도형은 단일 고채도 색상으로 채움. 테두리 없음. 그림자는 같은 색의 어두운 톤으로 최소한만 사용.
- **배경**: 어두운 딥 스페이스 색상(#1B2838)의 단색 배경. 미세한 별 입자 3-4개 허용.
- **코너/엣지**: 모든 형태에 큰 라운딩 적용. 날카로운 모서리 없음. 원, 캡슐, 둥근 사각형이 기본 도형.
- **연결선**: 시안색 빛나는 입자 흐름(glowing particle stream)으로 데이터 교환 표현. 전통적 화살표 사용하지 않음.
- **시각 장식**: 빛나는 글로우 효과, 작은 빛 입자, 반투명 원형 보케, 에너지 파동.
- **타이포그래피**: 텍스트 없음. 이 이미지에는 어떤 텍스트도 렌더링하지 않는다.
- **공간 구성**: 풀블리드 장면. 전경-중경-후경 3레이어 깊이감. 핵심 요소는 삼분법에 따라 배치. 모든 핵심 시각 요소는 캔버스 가장자리에서 5% 이상 안쪽에 완전히 포함.
- **시각 메타포**: 복잡한 개념을 일상적 장면이나 판타지 세계로 번역. 추상 개념을 물리적 사물로 의인화.

### Content Placement
- 화면 중앙에 거대한 둥근 공장 건물이 자리하고, 내부가 투명하게 보인다
- 공장 안에는 세 개의 둥근 체형 로봇 캐릭터가 각각 다른 작업을 하고 있다: 하나는 컨베이어 벨트 위의 부품을 조립하고, 하나는 빛나는 태블릿 형태의 대시보드를 확인하며, 하나는 시안색 에너지 빔으로 품질 검사를 수행한다
- 공장 외부 좌측에는 밀짚모자를 쓴 둥근 인간 캐릭터가 작은 모니터 화면을 보며 미소 짓고 있다
- 공장 상단에서 시안색 빛 입자들이 위성 안테나를 통해 하늘로 올라가며 데이터 전송을 표현한다
- 후경에 어두운 우주 배경 위로 작은 별과 보케가 흩어져 있다
- 전경 하단에 녹색으로 빛나는 작은 식물들이 공장 주변에서 자라나고 있어 지속가능성을 암시한다

- 비율: 16:9

### Background Treatment

### Color Palette
- 주조: #1B2838 (딥 스페이스) — 배경, 깊이감
- 보조: #00BCD4 (일렉트릭 시안) — 기술 요소, 데이터 흐름, 연결선, 로봇의 눈과 관절 글로우
- 강조: #FF6F00 (비비드 오렌지) — 핵심 포인트, 에너지, 조립 중인 부품의 열기
- 전경: #E0F7FA (아이스 화이트) — 전경 요소, 하이라이트, 인간 캐릭터의 밝은 옷

### Layout Structure
Central (중심) — 거대한 공장 건물을 중심에 두고 인간 캐릭터, 데이터 흐름, 식물 등 주변 요소가 둘러싼 장면

### Typography
텍스트 없음
