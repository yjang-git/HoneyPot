# INSTRUCTION

### Image Purpose
스마트 팩토리 기술 혁신 추진 전략을 체계적으로 보여주는 정부/공공기관 발표용 PPT 슬라이드

### Target Audience
정부 관계자, 정책 입안자, 공공기관 의사결정자

### Key Message
AI 기반 스마트 팩토리 혁신을 위한 3대 핵심 전략과 기대 효과

### Visual Style
gov 스타일 — 정부 백서처럼 굵은 테두리 박스에 번호가 매겨진 체계적 격자. 공식 문서와 발표 자료에 적합한 신뢰감 있는 디자인.

### Rendering Style
- **서피스**: 2px 실선 테두리 박스 + 흰색 채움. 헤더는 주조색 배경 + 흰색 텍스트. 본문은 흰색 배경 + 검정 텍스트.
- **배경**: 라이트 그레이(#F5F7FA) 전체 배경. 상단에 주조색(#1E3A5F) 배너. 하단에 가는 구분선.
- **코너/엣지**: 완전 직각 (0px radius). 정부 문서의 격식.
- **연결선**: 2px 실선 화살표 + 채워진 삼각형(▶).
- **시각 장식**: 채워진 아이콘, 단색 막대 그래프. 전통적 인포그래픽 스타일.
- **타이포그래피**: 3계층 (대/중/소). 좌측 정렬. 번호가 매겨진 리스트.
- **공간 구성**: 엣지 투 엣지 격자 + 20% 여백. 2×2 그리드.

### Content Placement
- 상단 배너(#1E3A5F 배경)에 '스마트 팩토리 혁신 추진 전략' 텍스트를 흰색 대형 텍스트로 배치
- 상단 배너 바로 아래 우측에 '산업통상자원부 · 2026' 텍스트를 소형으로 배치
- 상단 배너 아래 중앙에 'AI 기반 제조 혁신으로 생산성 40% 향상 달성' 텍스트를 중형으로 배치
- 하단 2/3 영역에 2×2 그리드(4개 박스) 배치:
  - 좌상 박스: 헤더에 '01 지능형 공정 자동화' 텍스트, 본문에 '실시간 품질 예측', '자율 공정 제어', '불량률 50% 감소' 텍스트
  - 우상 박스: 헤더에 '02 디지털 트윈 구축' 텍스트, 본문에 '가상 시뮬레이션', '설비 예지 보전', '다운타임 60% 절감' 텍스트
  - 좌하 박스: 헤더에 '03 데이터 기반 의사결정' 텍스트, 본문에 '통합 대시보드', '실시간 KPI 모니터링', '의사결정 속도 3배 향상' 텍스트
  - 우하 박스: 헤더에 '기대 효과' 텍스트를 오렌지(#E07B39)로, 본문에 '생산성 40% 향상', '불량률 50% 감소', '운영비용 30% 절감' 텍스트
- 4개 박스 사이에 2px 실선 화살표(▶)로 연결: 좌상→우상→좌하 순서, 세 박스 모두 우하 기대효과 박스를 향해 화살표

# CONFIGURATION

- 해상도: 3840 x 2160 (4K)

- 강조: #E07B39 (오렌지) — 핵심 성과 수치, 기대효과 박스 헤더
- 배경: #F5F7FA (라이트 그레이) — 전체 배경

### Layout Structure
Group (그룹) — 2×2 그리드로 4개 전략/효과 박스를 배치하고 화살표로 연결

### Typography
- 대형: 상단 배너 제목 (흰색, 굵게)
- 중형: 부제목, 박스 헤더 (주조색 또는 흰색, 굵게)
- 소형: 본문 항목, 기관명 (검정 또는 회색)
- 모든 텍스트 좌측 정렬
- 번호 배지(01, 02, 03)는 주조색 원형 배경 + 흰색 텍스트
