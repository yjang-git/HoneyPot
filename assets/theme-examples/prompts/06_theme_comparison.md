# INSTRUCTION

### Image Purpose
전통 제조 방식과 AI 스마트 팩토리 도입 후의 극적인 차이를 좌우 분할 화면으로 시각적으로 증명하는 비교 슬라이드

### Target Audience
제조업 경영진, 공장 관리자, 기술 도입 의사결정자

### Key Message
AI 스마트 팩토리 도입 전후의 압도적 차이

### Visual Style
comparison 스타일 — IMAX 분할 화면처럼 좌우 풀블리드 이미지 위에 핵심 수치가 떠 있다. 두 상태를 나란히 배치하여 차이점을 시각적으로 증명.

### Rendering Style
- **서피스**: 텍스트는 반투명 오버레이 위에만 배치. 이미지가 95%, 텍스트 오버레이가 5%. 좌측(Before)은 어둡고 복잡한 톤, 우측(After)은 밝고 정돈된 톤.
- **배경**: 좌측 전체에 어둡고 혼잡한 전통 공장 장면(차콜 톤). 우측 전체에 밝고 정돈된 스마트 공장 장면(에메랄드 톤). 각각 풀블리드로 화면 절반씩 차지.
- **코너/엣지**: 완전 직각(0px radius). 엣지 투 엣지 이미지 채움.
- **연결선**: 중앙에 대형 방향 화살표(→) 또는 VS 배지 1개만.
- **시각 장식**: 최소한. ✓/✗ 아이콘은 핵심 변화점에만. 변화율 배지(+45%, -92%). 중앙 화살표/VS 배지.
- **타이포그래피**: 2계층만 (제목/핵심 수치). 흰색 또는 밝은 텍스트. 반투명 오버레이 위에 배치.
- **공간 구성**: 50:50 좌우 분할. 패널 없이 풀블리드 이미지. 하단 30%에 반투명→반투명 블랙/화이트 그라데이션 오버레이.

### Content Placement
- 좌측 절반 전체에 어둡고 혼잡한 전통 공장 일러스트레이션(차콜 톤 #2D3436 기반): 수동 조립 라인, 종이 서류, 형광등 조명, 복잡하게 얽힌 배선
- 우측 절반 전체에 밝고 정돈된 스마트 공장 일러스트레이션(에메랄드 톤 #00B894 기반): 로봇 팔, 디지털 대시보드, LED 조명, 깔끔한 자동화 라인
- 정중앙에 큰 화살표(→) 배지 하나
- 좌측 하단 반투명 오버레이 위에 'Before' 텍스트와 '불량률 8.5%' 텍스트, '수동 검사' 텍스트를 배치
- 우측 하단 반투명 오버레이 위에 'After' 텍스트와 '불량률 0.3%' 텍스트, 'AI 자동 검사' 텍스트를 배치
- 우측 상단 영역에 코랄 레드(#FF7675) 변화율 배지로 '-96%' 텍스트를 강조 배치

### Background Treatment

### Typography
- 제목: Before/After 라벨, 흰색, 굵게
- 핵심 수치: 불량률 수치, 흰색 대형, 반투명 오버레이 위

- 보조: 수동 검사/AI 자동 검사, 흰색 소형
